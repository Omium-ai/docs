## Quick Start

Get up and running in minutes:

1. **Install the SDK**
   ```bash
   pip install omium
   ```

2. **Initialize your environment**
   ```bash
   omium init
   ```

3. **Create your first workflow**
   ```bash
   omium init-workflow --type crewai
   ```
